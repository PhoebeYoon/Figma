## 🍑 게슈탈트 이론 (GESTALT). 
**막스 베르트하이머 (Max Wertheimer) – “우리의 뇌는 구성 요소들을 개별적으로 보기에 앞서서 그 윤곽이나 패턴, 형태적 차이를 먼저 파악하려는 습성을 지녔다” 즉, 사람이 형태나 모양을 볼 때 인지하는 방식을 설명**
유사성의 원리, 근접성의 원리 , 공통의 원리, 연속성의 원리

- 유사성의 원리 : 색 모양, 크기가 동일한 요소들을 하나의 관계로 묶어서 보려는 경향. 유사한 형태끼리는 연관되어 보이는 원리, 생김새의 차이가 클수록 사람들은 더 빨리 대상을 구분한다
- 근접성의 원리 : 서로 이웃해 있는 요소들과 그렇지 않은 요소들을 구분하려는 경향 즉, 그룹핑이 되어 있는 요소들과 그렇지 않은 요소들을 구분하는 것으로 서로 연관되어 있는 정보들은 최대한 가깝게 배치하고 그렇지 않는 요소들은 떨어뜨려서 배치한다
- 공통의 원리 : 근접성의 원리와 비슷. 박스안에 있는 요소들을 같은 정보로 묶어서 보는 경향을 말한다.  박스형식의 카드형태를 생각하면 된다. 섹션를 박스별로 묶어서 보여주는 것
- 연속성의 원리 : 연속성을 띠는 배열을 관련성이 있다고 보는 것이다.
