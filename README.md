## 🍑 폰트 Variable

기존의 프로그램에서 폰트의 크기를 **size**로  굵기를 **weight**로 지정하였다.
그런데 피그마에서는 이에 덧붙여 variable이라는 것이 추가되었는데 아래와 같은 방법으로 확인할 수 있다.
