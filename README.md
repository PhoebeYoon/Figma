## 🍑 피그마 칼러설정 
맥과 윈도우에서 제작물이 동일한 색상으로 보이게 하기 위해 color space를 맞추도록 합니다. 또한 이렇게 해야 나중에 이미지로 디자인내용을 exporting 할때 색상의 차이가 발생하지 않습니다.   
color space에는 sRGB 와 unmanaged가 있는데 모두 **sRGB**로 설정해 줍니다
